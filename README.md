# user-management-cli
This is a simple Java command line tool that can be used to register, login and list users.

It is written using the following stack.
- Maven
- Spring Shell
- Spring Boot
- My SQL

## To run the application

`mvn spring-boor:run`

## To see the available commands

`shell:>help`

## To save user

`shell:>create-user`

## To list users

`shell:>list-users`
